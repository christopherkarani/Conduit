# Conduit (中文)

[English](../README.md) | [Español](README.es.md) | [日本語](README.ja.md) | [中文](README.zh-CN.md)

**只要做一点配置调整，就可以在 Claude、GPT-4o、Apple Silicon 上的本地 Llama，以及 Apple 的基础模型之间切换。**

Conduit 是一个 Swift 框架，用一套 API 同时处理云端和本地语言模型。

## 核心特性

- **快速:** 针对 Apple Silicon 进行优化，提供对 MLX 的一流支持。
- **类型安全:** 使用 Swift 宏，在编译时保证结构化输出。
- **统一内存:** 利用 Apple 的统一内存架构进行高效的本地推理。
