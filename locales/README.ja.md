# Conduit (日本語)

[English](../README.md) | [Español](README.es.md) | [日本語](README.ja.md) | [中文](README.zh-CN.md)

**Claude、GPT-4o、Apple Silicon上のローカルLlama、AppleのFoundationモデルを、小さな設定変更だけで切り替えられます。**

Conduitは、クラウドとローカルの言語モデルを1つのAPIで扱うためのSwiftフレームワークです。

## 主な特徴

- **高速:** Apple Silicon向けに最適化され、MLXの最高クラスのサポートを提供。
- **型安全:** Swiftマクロを使用して、コンパイル時に保証された構造化出力を実現。
- **ユニファイドメモリ:** Appleのユニファイドメモリ・アーキテクチャを活用し、効率的なローカル推論を可能に。
