# Thread — Why iOS Developers Need Conduit — 2026-03-25

## 1/ (Hook)

iOS devs: you shouldn't need 5 different SDKs to talk to AI models.

The problem isn't picking a provider. It's rewriting everything when you switch.

Conduit: one protocol, 9 backends.

## 2/

Same API whether you're hitting:
- OpenAI / Anthropic / HuggingFace (cloud)
- MLX on Apple Silicon (local)
- Llama.cpp, CoreML, FoundationModels

Switching from GPT-4 to Claude = change one initializer.

## 3/

Tool calling is where things get ugly.

OpenAI uses function calling. Anthropic uses tool_use. Different formats everywhere.

You end up writing adapters just to book a meeting room.

## 4/

Conduit's ToolExecutor handles the whole loop:

```swift
session.toolExecutor = ToolExecutor(tools: [CalendarTool(), SearchTool()])
let response = try await session.send("Schedule a meeting with Sarah")

// Tools execute automatically. Retries included.
```

One send() call. That's it.

## 5/

Structured output used to mean manual Codable everywhere.

```swift
@Generable
struct WeatherResult {
    @Guide(description: "City name")
    var city: String
    var temperature: Double
}
```

One attribute. Macro synthesizes init, schema, partial JSON handling.

## 6/

Streaming JSON is painful.

Tokens arrive out of order. Objects get split across chunks. You debug partial responses more than you build features.

Conduit tries closing braces automatically. Falls back to string only when repair fails.

## 7/

Local inference on iOS? MLX on Apple Silicon makes it practical now.

```swift
config.runtimeFeatures.kvQuantization.enabled = true
try await provider.warmUp(model: .llama3_2_1b, prefillText: "Hi", maxTokens: 5)
```

No API keys. No latency. No sending data anywhere.

## 8/

For SwiftUI devs: ChatSession is @Observable.

Direct binding. Thread-safe via NSLock (never held across await).

```swift
ForEach(viewModel.session.messages) { message in
    MessageRow(message)
}
```

## 9/ (Closer)

The real pitch: treat AI as infrastructure, not vendor lock-in.

Providers come and go. Claude 4 will drop. GPT-5 will follow.

Your Conduit code keeps working.
